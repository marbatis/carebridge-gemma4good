# Video Upload Checklist

Official Kaggle requirements to satisfy:
- attach the video to the Media Gallery
- keep it at 3 minutes or less
- publish it to YouTube
- provide a direct link
- make sure judges can watch it without login

Practical checklist:
- export final video at 1080p
- upload as `Public` or `Unlisted` on YouTube
- confirm playback works in an incognito browser window
- copy the direct YouTube URL
- attach that URL in the Kaggle write-up Media Gallery
- add a strong cover image to the write-up
- add the public code repo under Attachments > Project Links
- add the live demo URL under Attachments > Project Links

Recommended file naming:
- `CareBridge - Gemma 4 Good Hackathon Demo.mp4`
- `CareBridge Cover.png`

Recommended YouTube title:
- `CareBridge: Local-First Maternal Health Navigation with Gemma 4`

Recommended YouTube description:
- one sentence on the problem
- one sentence on how Gemma 4 is used
- links to Kaggle write-up, code repo, and demo
